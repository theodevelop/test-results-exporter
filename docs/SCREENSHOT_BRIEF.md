# Captures à prendre pour les stores

Prendre de vraies captures depuis Firefox ou Chrome, sans données personnelles ni noms de projets confidentiels.

1. **Écran d’accueil** : popup ouverte sur une page de résultats compatible, avec les deux choix « Tous les tests » et « Échecs uniquement ».
2. **Avancement** : popup pendant l’export, avec la barre de progression non vide et le nom du test en cours.
3. **Succès** : popup après export, avec le message indiquant le nombre de tests exportés et le nombre d’échecs.

Recommandations :

- cacher ou flouter les informations personnelles visibles dans la page ;
- garder les captures lisibles ;
- ne pas utiliser une image de démonstration qui ferait croire à une fonctionnalité absente ;
- privilégier une fenêtre de navigateur en thème clair ou sombre propre et cohérent.
