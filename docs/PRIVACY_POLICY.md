# Politique de confidentialité — Test Results Exporter

**Dernière mise à jour : 24 juin 2026**

## Résumé

Test Results Exporter ne collecte, ne vend et ne transmet aucune donnée personnelle ou donnée de navigation vers un serveur externe.

## Données traitées localement

Lorsque l’utilisateur déclenche un export, l’extension lit uniquement les éléments **déjà affichés** dans l’onglet actif : noms de tests, catégories, statut visible et messages d’échec visibles.

Ces informations servent uniquement à créer un fichier JSON téléchargé localement par le navigateur.

## Données non collectées

L’extension ne :

- crée aucun compte ;
- n’utilise aucun cookie, outil d’analyse ou publicité ;
- n’envoie aucune donnée à un serveur ;
- n’accède pas à l’historique, aux mots de passe, aux requêtes réseau ou aux autres onglets ;
- ne récupère pas de contenu non affiché par la page.

## Permissions

- `activeTab` : permet d’agir uniquement dans l’onglet actif après une action explicite de l’utilisateur.
- `scripting` : permet de lire les éléments déjà rendus dans cette page pour construire le fichier JSON.

## Conservation

L’extension ne conserve aucune donnée. Le fichier JSON est créé localement dans le dossier de téléchargement défini par l’utilisateur dans son navigateur.
