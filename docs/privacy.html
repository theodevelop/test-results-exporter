Privacy Policy

Last updated: 24 June 2026

Summary

Test Results Exporter does not collect, sell, or transmit personal data or browsing data to an external server.

Data processed locally

After the user starts an export, the extension reads only elements already rendered in the active tab, such as test names, categories, visible status, and visible failure messages. This information is used only to create a JSON file downloaded locally by the browser.

Data not collected

  • No user account is created.
  • No cookie, analytics service, advertising SDK, or remote server is used.
  • The extension does not access browser history, passwords, network requests, or other tabs.
  • The extension does not retrieve content that is not displayed by the page.

Permissions

  • activeTab: allows the extension to act only in the current tab after an explicit user action.
  • scripting: allows the extension to read already-rendered page elements to build the local JSON export.

Retention

The extension does not retain any data. The generated JSON file is stored locally in the download location configured by the user in their browser.

Publisher and contact

Test Results Exporter is published by theodevelop. For privacy-related questions or feedback, please open an issue in the project repository.
